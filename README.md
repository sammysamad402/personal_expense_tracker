# 💰 Personal Expense Tracker

A Python-based Expense Tracker application that allows users to record daily expenses, manage monthly budgets, and maintain financial records efficiently using CSV file storage.

This project demonstrates core Python programming concepts including file handling, data validation, modular programming, and testing.

---

## 🚀 Features

- 📌 Add and store expenses
- 📌 Categorize expenses
- 📌 View expense history
- 📌 Set monthly budget limits
- 📌 Track spending against budget
- 📌 Persistent CSV data storage
- 📌 Input validation and error handling
- 📌 Unit testing support
- 📌 Interactive command-line interface

---

- Jupyter Notebook
